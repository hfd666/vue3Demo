# 组件示例页面说明

## 页面列表

项目已创建三个独立的组件示例页面，用于展示封装组件的使用方法。

### 1. 表格示例页面

- 路由：`/table-demo`
- 文件：`src/views/demos/TableDemo.vue`
- 图标：Grid（表格）
- 展示内容：
  - ProTable 表格组件
  - SearchForm 查询表单
  - useTable Hook 的使用
  - 完整的 CRUD 操作示例

### 2. 弹窗示例页面

- 路由：`/dialog-demo`
- 文件：`src/views/demos/DialogDemo.vue`
- 图标：MessageBox（消息框）
- 展示内容：
  - 标准表单弹窗
  - 只读弹窗（无底部）
  - 确认提示弹窗
  - 自定义底部弹窗
  - 全屏弹窗
  - 大尺寸弹窗
  - 表单验证示例

### 3. 上传示例页面

- 路由：`/upload-demo`
- 文件：`src/views/demos/UploadDemo.vue`
- 图标：Upload（上传）
- 展示内容：
  - 单图上传
  - 多图上传
  - 单文件上传
  - 多文件上传
  - 查看模式（disabled）
  - 文件类型和大小限制

## 菜单结构

左侧菜单已简化为：

```
├── 首页 (/)
├── 表格示例 (/table-demo)
├── 弹窗示例 (/dialog-demo)
└── 上传示例 (/upload-demo)
```

## 访问方式

启动项目后，可以通过以下方式访问：

1. 点击左侧菜单对应的菜单项
2. 直接访问 URL：
   - http://localhost:5173/table-demo
   - http://localhost:5173/dialog-demo
   - http://localhost:5173/upload-demo

## 页面特点

每个示例页面都包含：

1. **实际演示**：可交互的组件示例
2. **功能说明**：列出组件的核心特性
3. **代码示例**：展示基本的使用代码
4. **多种场景**：覆盖常见的使用场景

## 已删除的页面

以下页面和路由已被删除：

- Dashboard（仪表盘）及其子页面
- User（用户管理）及其子页面
- System（系统管理）及其子页面
- Docs（开发文档）及其子页面

对应的视图文件仍保留在 `src/views/` 目录中，如需使用可以重新添加路由。

## 注意事项

1. **上传功能**：由于是示例项目，上传功能需要配置真实的后端接口
2. **数据模拟**：表格数据使用模拟数据，实际项目中需要对接真实 API
3. **图标注册**：已在 `src/main.ts` 中注册了示例页面使用的图标
